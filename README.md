# C_reddot

# 簡化版撿紅點

## 遊戲進程

- **遊戲人數**：2～4 人  
- **起始手牌**：每位玩家擁有 `24 ÷ 玩家數量` 張手牌  
- **初始檯面（海底）**：4 張牌  

遊戲採 **輪流制** 進行，直到所有玩家的手牌用完為止。

### 回合規則
- 每位玩家回合時：
  - 若手牌中有 **可與海底牌配對** 的牌，**必須出牌配對**
  - 若 **無法配對**，則將一張手牌放入海底

---

## 配對規則

- **牌面數字 < 10**
  - 手牌數字 + 海底牌數字 = **10** → 配對成功
- **牌面數字 ≥ 10**
  - 手牌數字 = 海底牌數字 → 配對成功

---

## 計分規則

玩家得分依 **花色與數字** 計算：

### 🔴 紅色花色（紅心 ♥、方塊 ♦）
- 數字 **> 9** → **10 分**
- 數字 **= 1（A）** → **20 分**
- 其他數字 → **對應數字分數**

### ⚫ 黑色花色（黑桃 ♠、梅花 ♣）
- **黑桃 A** → **10 分**
- 其餘牌 → **不加分**
