# C_reddot

簡化版撿紅點

/n
遊戲進程：

遊戲人數2~4人

每位玩家初始擁有 24/玩家數量 張手牌

初始檯面(海底) 4張牌

遊戲採輪流進行，直到所有玩家手牌用完為止。

每個玩家回合時，若玩家手上有能和海底牌配對的牌，則必須出牌配對；若無法配對，則將一張手牌放進海底。

 
配對規則：

在牌面數字小於10的情況下，手牌數字與海底牌數字加總為10即配對成功。

在牌面數字大於等於10的情況下，手牌數字與海底數字相同即配對成功。

 
計分規則：

玩家得分依照牌的花色與數字計算

 紅色花色（紅心、方塊）

  數字 > 9 → 10 分

  數字 = 1 (A) → 20 分

  其他數字 → 對應數字分數



  黑色花色（黑桃、梅花）

  黑桃Ace → 10 分

  其餘不加分
